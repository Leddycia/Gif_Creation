# GIF Creator

A simple Python script to create animated GIFs from a collection of images using the `imageio` library.

## Project Structure

- `main.py`: The main script that processes images and generates the GIF.
- `team.gif`: The output GIF generated by the script.
- Various image files (`.png`, `.jpg`): Source images used for GIF creation.

## Prerequisites

Before running the script, ensure you have Python installed on your system. You will also need to install the `imageio` library.

### Dependencies

Install the required library using pip:

```bash
pip install imageio
```

## Usage

1. Place your source images in the project directory.
2. Open `main.py` and update the `filenames` list with the names of your image files.
3. Run the script:

```bash
python main.py
```

4. The generated GIF will be saved as `team.gif` in the project root.

## Script Overview

The `main.py` script performs the following steps:
1. Imports the `imageio` library.
2. Defines a list of input image filenames.
3. Reads each image into a list using `imageio.imread()`.
4. Saves the collection of images as a GIF using `imageio.mimsave()` with a specified duration for each frame.

## Features

- **Easy to use**: Simple configuration in `main.py`.
- **Fast**: Quickly converts images to GIF format.
- **Customizable**: Adjust frame duration and input images easily.
